# SCT_WD_T1
COLOUR CHANGING NAVIGATION MENU

Overview

This is a personal portfolio website designed to showcase the skills, projects, and contact information of Pratik Anand Khot, a Front-End Developer and Information Science student. The website features a modern glass-glow aesthetic with subtle animations and interactive elements, providing a futuristic and professional look.

Features

Home Section:
Introduction with text aligned to the left.
Profile photo with circular glass-glow effect and hover animation.

About Section:
Fully centered content describing personal interests and technical skills.

Projects Section:
Project cards with circular images and hover effects.
Each card displays project title and description.

Contact Section:
Email link and social media buttons (GitHub, LinkedIn).
Centered layout for easy access.

Design & Aesthetics:
Glass-glow effect using backdrop-filter and semi-transparent backgrounds.
Smooth hover effects for buttons and images.
Futuristic gradient background with subtle animation.

Responsive Design:
Fully responsive layout for desktops, tablets, and mobile devices.
Sections adapt to smaller screens with stacked layouts.

Technologies Used:
HTML5
CSS3 (Flexbox, Glassmorphism, Animations)
AOS (Animate on Scroll) Library
Google Fonts (Outfit)
